Provost.views.provost(request)[source]

View function to handle the provost view.

This function retrieves the provost object based on the logged-in user and the associated hall. It then gets all rooms in the hall and creates a dictionary to store room availability information. The function processes allocation, adding student to room, and removing student from room form submissions.
