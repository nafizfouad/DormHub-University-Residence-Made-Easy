Student.views.student(request)[source]

View function to handle the student view.

This function retrieves the student object based on the logged-in user’s email. It gets the hall and room associated with the student and retrieves repair requests for the student’s hall. The function processes form submissions for requesting room change and repair.

Parameters:

request (HttpRequest) – The HTTP request object.

Returns:

The HTTP response object rendering the student template.
